## Directory Summary
The `synth_data_forge` directory is the root of the synthetic data generation engine for the ScholarConnect project. It contains the primary entry point for the generation script and orchestrates the creation of a complete, relational synthetic dataset including students, professors, courses, and their interactions.

## Documentation Index
- [`__init__.py`](../docs/synth_data_forge_docs/__init___doc.md) — Public API for the ScholarConnect synthetic-data package.
- [`main.py`](../docs/synth_data_forge_docs/main_doc.md) — Entry point script that orchestrates the full data generation workflow.
