## Directory Summary
The `data` directory contains the output of the synthetic data generation engine. These CSV files follow the ScholarConnect database schema and are populated with realistic records for students, instructors, courses, enrollments, and evaluations.

## Dataset Index
- `STUDENT.csv` — 80 records of synthetic student data.
- `INSTRUCTOR.csv` — 12 records of synthetic instructor data.
- `COURSE.csv` — Complete catalogue of 25 courses.
- `ENROLLMENT.csv` — Student enrollment records across various courses.
- `EVALUATION.csv` — Feedback and ratings for instructors.
